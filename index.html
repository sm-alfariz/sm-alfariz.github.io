Politikus itu Abstrak

sumber php.net : Classes defined as abstract may not be instantiated, and any class that contains at least one abstract method must also be abstract. Methods defined as abstract simply declare the methodâ€™s signature - they cannot define the implementation.

Abstract Class dilihat dari penamaanya dapat kita katakan bahwa itu adalah hal yang Abstract alias tidak nyata (tidak jelas). Abstract Class biasanya dibuat untuk diturunkan ke Class lain dengan kata lain Abstract Class akan berisi semua hal paling general dari Class-Class anaknya. dari situ lah politkusi saya turunkan, karena bila politikus saja bisa saja ada politikus yg baik dan macam maca tergantung diturunkan kepada class anaknya, Seperti Class pada umumnya, Abstract Class juga bisa mempunyai properties,hak akses,dan method namun ###Abstract Class tidak bisa dibuatkan Object nya sendiri.###

berikut contoh class abstrak dari politikus: AbstractPolitikus.php

Kita membutuhkan class lain yang akan meng extends class di atas PolitikusBaik.php

Seperti dilihat padsa class atas/induk nya ada setter/getter method class tpi di kerjakan oleh klass anaknya, ini sudah menjadi hal yg biasa untuk class di oop terus di mana sisi bedanya dengan class Abstract
