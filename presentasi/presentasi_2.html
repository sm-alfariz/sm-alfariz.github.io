Apa itu Vibe Coding?

Pergeseran cara membangun software: dari menulis sintaks, menjadi mengarahkan AI.

◂ Dulu / Traditional
✕ Mengetik sintaks manual, debugging pusing.
✕ Membaca dokumentasi tebal sebelum mulai.
✕ Error berantai karena satu salah sintaks.
▸ Sekarang / Vibe Coding
✓ Menulis instruksi dalam bahasa alami.
✓ Review hasil kode yang ditulis AI.
✓ Uji coba instan, langsung terlihat hasilnya.
💡 Dipopulerkan oleh Andrej Karpathy (Awal 2025) • Peran manusia naik kelas menjadi "Project Manager"
Referensi

Beberapa istilah dalam Vibe Coding

Istilah-istilah penting yang perlu dipahami dalam ekosistem Vibe Coding.

Intent-Driven Development

Fokus pada "apa" yang ingin dibuat, bukan sintaks kodenya.

System Prompt

Instruksi awal untuk mengatur konteks, gaya, dan aturan kerja AI.

Few-Shot Prompting

Memberikan contoh pola atau format sebelum meminta tugas inti.

Edge Cases

Kondisi ekstrem atau error tidak terduga yang perlu diantisipasi AI.

Codebase

Kumpulan file dan seluruh struktur program yang dikelola.

Illusion of Competence

Fenomena merasa sangat ahli karena proses coding terasa terlalu instan.

Prompting

Seni Memberi Perintah

Formula prompt efektif: Role → Goal → Features → Style.

[ROLE] "Bertindaklah sebagai Senior Web Developer... [GOAL] ...buatkan saya aplikasi Kalkulator Anggaran Harian. [FEATURES] ...Fiturnya harus bisa input uang saku, simpan di LocalStorage. [STYLE] ...Gunakan desain modern minimalis tema terang. ▌ menunggu respon AI...
